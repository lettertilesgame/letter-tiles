    Einrichtung (einmalig) — Supabase-Tabelle & Zugang

    1. Kostenloses Projekt auf supabase.com anlegen.
    2. supabase-config.js öffnen und url + anonKey eintragen (Project Settings → API).
    3. Im Supabase SQL-Editor dieses SQL einmal ausführen: 

    create table if not exists public.lt_students (
  id uuid primary key default gen_random_uuid(),
  class_code text not null,
  student_id text not null,
  name text not null,
  levels_passed int default 0,
  total_levels int default 0,
  current_lesson int default 1,
  current_level int default 1,
  accuracy int default 0,
  updated_at timestamptz default now(),
  unique (class_code, student_id)
);

alter table public.lt_students enable row level security;
create policy "anon read"   on public.lt_students for select to anon using (true);
create policy "anon insert" on public.lt_students for insert to anon with check (true);
create policy "anon update" on public.lt_students for update to anon using (true) with check (true);

-- Live-Updates aktivieren:
alter publication supabase_realtime add table public.lt_students;

    Hinweis Datenschutz: Mit diesen offenen Regeln kann jede Person mit Klassencode + öffentlichem anon-Key die Klassendaten lesen/schreiben. Nutze nur Vornamen/Spitznamen und keine sensiblen Daten.
